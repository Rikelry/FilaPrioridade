# FilaPrioridade

## Membros
- @Rikelry | Rikelry Monteiro Souza

## Descrição
Este projeto implementa um Tipo de Dado Abstrato (TDA) de Fila com Prioridade em TypeScript.
A estrutura possui cinco níveis de prioridade (muito baixa, baixa, normal, alta e muito alta),
respeitando sempre a ordem de prioridade no momento da remoção dos elementos.
A fila possui capacidade máxima definida no momento de sua criação.

## Como executar
1. Clone o repositório:
```
git clone https://github.com/Rikelry/FilaPrioridade.git
```

2. Instale as dependências:
```
npm install
```

3. Execute o projeto:
```
npm run start
```

4. Abra o navegador, aperte ```F12``` e visualize o resultado no ```console```.
